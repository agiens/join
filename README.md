# Agiens Coding Challenges

Добро пожаловать в Agiens! Мы верим в практические задания по программированию, которые проверяют навыки, нужные в реальной работе.

Задача: реализовать агента OpenClaw, который будет разворачивать OpenClaw через общение в чате.

1. [Step 1: Развернуть OpenClaw с помощью Docker](./challenges/step1)
2. [Step 2: Сделать инструмент, который автоматизирует Step 1](./challenges/step2)
3. [Step 3: Подключить инструмент из Step 2 к OpenClaw](./challenges/step3)

При желании можно использовать: [OpenCode](https://github.com/code-yeongyu/oh-my-opencode) (но это не обязательно. Вы можете применять любой удобный вам AI-workflow и любые инструменты, к которым вы привыкли)
